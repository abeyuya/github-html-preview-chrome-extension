相対パスのアセット解決サンプル

この HTML は ./logo.png を 相対パスで参照しています。 拡張機能はこれを raw.githubusercontent.com 配下の URL に自動解決し、 画像が表示されます。

上にロゴ画像が表示されていれば、./logo.png が相対パスから 正しく解決・読み込みできています。

※ <img> など画像は相対解決後そのまま表示されますが、 <link rel="stylesheet"> や <script src> で読み込む CSS / JS は raw が text/plain + nosniff で返すためブラウザが 適用を拒否します。スタイルや動的処理はこのページのようにインラインで記述するか、 CDN の絶対 URL を利用してください（Swagger UI サンプル参照）。
